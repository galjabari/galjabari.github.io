Install Proxy Server on Ubuntu

Update package lists and install Squid proxy server.

sudo apt update
sudo apt install squid

Edit the Squid configuration file using a text editor.

sudo nano /etc/squid/squid.conf

Inside the file, locate the acl localnet section and uncomment the following line:

http_access allow localnet

To apply the changes made to the configuration file, restart the Squid service.

sudo systemctl restart squid.service

Check the status of the Squid service to ensure it restarted without errors.

sudo systemctl status squid.service

You can now connect to this proxy server from your client devices. Enter the IP address of your proxy server running Squid and the port number (default port for Squid is 3128) on the proxy settings of your device.

Once you've configured the proxy settings on your device, it should route all internet traffic through the Squid proxy server.

To monitor access to your proxy server in real-time, you can tail the access log.

sudo tail -f /var/log/squid/access.log

To block specific websites, edit the Squid configuration file again.

sudo nano /etc/squid/squid.conf

Add the following lines to block Facebook and YouTube, ensure to place deny ACL above ACLs that allow network access:

acl blocklist dstdomain .facebook.com .youtube.com
http_access deny blocklist

After updating the configuration, reload Squid service for changes to take effect.

sudo systemctl reload squid.service
