Create containers using Docker Compose

To clone my repository, run the following command:

git clone https://github.com/galjabari/cloud-computing.git

To run my hands-on labs, open Visual Studio Code using the following commands:

cd cloud-computing
code .

For example, to create and run WordPress stack with Docker Compose, you can run the following commands:

cd wordpress
docker compose up

Compose pulls images from Docker Hub, creates containers, and start the services. You can log into WordPress by opening a web browser and going to:

http://localhost:8080

Replace localhost with the IP address of your machine if needed.
