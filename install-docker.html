Install Docker on Ubuntu

To get started quickly, you can install Docker using the following command

 curl -sSL https://galjabari.github.io/install-docker.sh | bash

To download the bash script before installation, you can run the following commands:

 wget -O install-docker.sh https://galjabari.github.io 
 sudo bash install-docker.sh

Note: It may be necessary to restart your machine for changes to take effect.
