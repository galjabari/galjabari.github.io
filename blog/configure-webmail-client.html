Configure Webmail client on Ubuntu

This guide assume that you have an existing mail server with the hostname mail.example.com.

First, install Docker on your server to deploy Roundcube webmail using the following script:

curl -sSL https://galjabari.github.io/install-docker.sh | bash

Next, run the following command to install and configure Roundcube Docker container:

docker run -d \
  --name roundcube \
  -e ROUNDCUBEMAIL_DEFAULT_HOST=mail.example.com \
  -e ROUNDCUBEMAIL_DEFAULT_PORT=143 \
  -e ROUNDCUBEMAIL_SMTP_SERVER=mail.example.com \
  -e ROUNDCUBEMAIL_SMTP_PORT=25 \
  -e ROUNDCUBEMAIL_USERNAME_DOMAIN=example.com \
  -p 80:80 \
  roundcube/roundcubemail

This command will download the Roundcube image, create and start the container.

To access Roundcube webmail, open a web browser and enter your server's hostname or IP address.
