Install SSH Server on Ubuntu

Run the following command in your terminal to make sure your package lists are up to date:

sudo apt update

Next, install the OpenSSH server package by executing the following command:

sudo apt install openssh-server -y

Once the installation is complete, you can verify that the SSH server is running by checking its status. Run the command:

sudo systemctl status ssh.service

If the SSH server is active and running, you'll see a message indicating its status.

To connect to the SSH server, you'll need its IP address and your username. Assuming the IP address of your system is 192.168.1.5 and your username is ubuntu, you can connect using the following command:

ssh ubuntu@192.168.1.5

After entering the password, you should be logged in to your system via SSH.
