Create containers using Docker Compose

To clone my repository, run the following command:

git clone https://github.com/galjabari/cloud-computing.git

To run my hands-on labs, open Visual Studio Code with the following commands:

cd cloud-computing
code .

For example, to create and run a WordPress stack with Docker Compose, use these commands:

cd wordpress
docker compose up

Compose pulls images from Docker Hub, creates containers, and starts the services. You can log in to WordPress by opening a web browser and navigating to http://localhost:8080. Replace localhost with your machine's IP address if needed.
