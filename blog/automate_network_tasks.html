Automate network tasks using Python

To automate network device configuration such as making configuration changes and gathering output commands, you can use Netmiko library in Python.

First, ensure your network device supports SSH and has SSH enabled. For example, to setup SSH on a Cisco switch for remote management, execute the following commands:

Switch>en 
Switch#conf t
Switch(config)#hostname S1
S1(config)#ip domain-name example.com
S1(config)#crypto key generate rsa
S1(config)#ip ssh version 2
S1(config)#username admin privilege 15 secret pass@123
S1(config)#line vty 0 4
S1(config-line)#transport input ssh
S1(config-line)#login local
S1(config-line)#exit
S1(config)#interface vlan 1
S1(config-if)#ip address 192.168.1.254 255.255.255.0
S1(config-if)#no shut
S1(config-if)#exit
S1(config)#exit

Replace 192.168.1.254 with the IP address of your network device and admin with your desired username.

After configuring Cisco switch, create a folder for Python script on Linux system:

mkdir automation

Inside the folder, create a virtual environment and activate it:

cd automation
python -m venv .venv
source .venv/bin/activate

Install the Netmiko using package installer for Python:

pip install netmiko
